# Image Processing App

---
1. Install the dependencies
```bash
mkdir lib && cd lib
```
To build the project install these dependencies in the lib directory:
- imgui
- imgui-node-editor
- glfw

2. Making the project

```bash
mkdir build && cd build
cmake ..
make -j($numproc)
```

3. Run the App.
```bash
./ImageNodeApp
```
